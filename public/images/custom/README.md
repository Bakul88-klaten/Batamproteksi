# Taruh gambar Anda sendiri di sini

Situs otomatis mendeteksi file dengan nama berikut di folder ini —
tidak perlu ubah kode apa pun, cukup upload lalu deploy ulang:

| Nama file            | Dipakai di halaman         |
|-----------------------|-----------------------------|
| `home.jpg`             | Beranda                    |
| `mobil.jpg`            | Asuransi Mobil              |
| `properti.jpg`         | Asuransi Properti           |
| `liability.jpg`        | Public Liability            |
| `surety.jpg`           | Surety Bond                 |

Format yang didukung: `.jpg`, `.jpeg`, `.png`, `.webp` (pakai salah satu,
tidak perlu semuanya). Kalau lebih dari satu format untuk key yang sama
ada, urutan prioritas: jpg → jpeg → png → webp.

Rekomendasi:
- Rasio 5:2 (misalnya 1600×640 px) — gambar akan otomatis dipotong
  (center-crop) mengikuti rasio ini kalau rasio aslinya beda.
- Ukuran file di bawah ~500 KB per gambar supaya halaman tetap cepat dimuat.

Selama nama file belum ada, halaman akan tetap menampilkan ilustrasi
placeholder bawaan — jadi aman untuk diisi bertahap, halaman tidak akan
kosong atau rusak.
